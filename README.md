## ✨ Overview

Daruka helps creators, founders, entrepreneurs, and innovators present their ideas with confidence.

Whether you're preparing for investors, startups, competitions, or business meetings, Daruka leverages AI to streamline the pitching process and help you create professional-quality content effortlessly.

---

## 🚀 Features

* 🤖 **AI-Powered Assistance** – Generate and refine pitch content instantly.
* 🎨 **Modern UI/UX** – Beautiful dark-themed interface designed for productivity.
* ⚡ **Lightning Fast Performance** – Optimized for speed and responsiveness.
* 📱 **Fully Responsive** – Works seamlessly across desktop, tablet, and mobile devices.
* 🔒 **Secure Authentication** – Safe and secure user access.
* 🌙 **Elegant Design System** – Minimal, distraction-free experience.
* ☁️ **Cloud Hosted** – Accessible from anywhere.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/theshekhr/Daruka.git
```

Navigate into the project:

```bash
cd Daruka
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
